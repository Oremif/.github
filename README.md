# Oremif `.github`

This repository stores organization-level GitHub metadata for Oremif.

It includes:

- the organization profile README in [`profile/README.md`](profile/README.md);
- default community health files such as [`CONTRIBUTING.md`](CONTRIBUTING.md),
  [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md), [`SECURITY.md`](SECURITY.md),
  [`SUPPORT.md`](SUPPORT.md), and [`GOVERNANCE.md`](GOVERNANCE.md);
- a default pull request template used by Oremif repositories that do not provide
  their own.

Repository-specific files take precedence over these defaults.
